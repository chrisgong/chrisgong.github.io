运行DEMO

Demo Github地址: https://github.com/facebook/react-native

下载项目

配置Android环境并运行Demo

  1. 将下载的Examples用Android Studio打开. (如果本地没有gradle会在导入过程中下载, 需要很长时间, 不要着急, 耐心等待)
  2. 修改项目local.properties, 加入以下路径.
    • sdk.dir=$Android SDK路径
    • ndk.dir=$Android NDK路径
  3. 打开Terminal

  4. 启动本地资源服务

  5. 编译运行项目

配置iOS环境并运行Demo

  1. 启动本地资源服务

  2. 打开Examples/*/*.xcodeproj文件

  3. Build -> Run in Xcode

注意事项

  1. ndk版本不要高于r10b
  2. 脚本执行需要提供权限
  3. Node等支持工具需要最新版本
  4. build过程需要下载boost_1_57_0.zip(100+MB), 最好直接去下载完了扔到ReactAndroid/build/downloads下, 如果在线下载的话最好开墙. 否则你懂得.
  5. 过程中提示缺包的话, npm info查下, 然后install.

目前Demo的运行就这些坑, 没太多难度的.
